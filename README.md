## Getting Started

1. Run `npm install`
2. Copy `.env.example` to `.env` and configure your environment variables
3. Run `npm run dev`

## Environment Variables

This project uses environment variables for configuration. All environment variables used by Vite must be prefixed with `VITE_`.

### Setup

1. Copy the `.env.example` file to `.env`:
   ```bash
   cp .env.example .env
   ```

2. Update the values in `.env` with your configuration

### Available Variables

- `VITE_API_URL` - API endpoint URL
- `VITE_APP_NAME` - Application name
- `VITE_APP_VERSION` - Application version

### Usage in Code

Import and use environment variables through the config module:

```typescript
import env from './config/env'

console.log(env.apiUrl)
console.log(env.appName)
```

Or access directly:

```typescript
const apiUrl = import.meta.env.VITE_API_URL
```

**Note:** The `.env` file is ignored by git to keep sensitive data secure. Never commit `.env` files to version control.
